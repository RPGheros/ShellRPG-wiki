# ShellRPG-wiki · v0.7.6

## 1. Beschreibung

**Artefaktrolle:** Redigierte Dokumentations- und Wissensschicht für Lore, Systeme, Bedienung, Glossar, FAQ und Projektorientierung.

**Zweck:** Dieses Artefakt erklärt ShellRPG aus Spieler-, Designer- und Integrationssicht, ohne private Serverdetails offenzulegen. Es funktioniert wie eine projektinterne Wikipedia mit Inhaltsseiten, Querverweisen und Navigationsstruktur.

**Verknüpfte Artefakte:**
- `ShellRPG-server` bleibt privat und wird hier nur abstrahiert beschrieben.
- `ShellRPG-client` und `ShellRPG-www` werden aus Bedien- und Integrationssicht dokumentiert.
- Die Wiki ist die zentrale öffentliche Navigationshilfe über Spielwelt, Mechaniken und Begriffe.

**Governance:** `WIKI-REDACTED`

Die eigentliche Inhaltsnavigation beginnt in [`index.md`](index.md). Dort ist die Wikipedia-artige Navigationsstruktur mit logisch gruppierten Inhaltsseiten verlinkt.

## 2. Abhängigkeiten

- Markdown-Viewer oder GitHub/GitLab-Rendering
- optional lokaler Webserver für bequeme Navigation

## 3. Installation

Zum direkten Lesen reicht ein Markdown-Viewer oder ein Git-Hosting-Frontend.

Optional lokal im Browser:
```bash
python -m http.server 8090
```

Dann öffnen:
```text
http://127.0.0.1:8090/
```

## 4. Feedback & Contribution

Feedback sollte fehlende Querverweise, unklare Begriffe oder redaktionelle Inkonsistenzen benennen.
Contribution muss die Redaktionsgrenze respektieren: nützlich für Öffentlichkeit, aber ohne Rekonstruktion privater Kernlogik.
Neue Inhaltsseiten sollen in die Navigationsstruktur der Wiki aufgenommen werden.
