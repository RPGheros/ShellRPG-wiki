# Terminal-Guide

## Start

## Windows PowerShell

```powershell
.\.venv\Scripts\Activate.ps1
python -m shellrpg_client
```

## Bash / Zsh

```bash
source .venv/bin/activate
python -m shellrpg_client
```


## Live-Status

Der Terminal-Client pollt den Server live während laufender Aktionen wie `hunt`, `gather` und `explore`.

## Wichtige Befehle

- `hunt N`
- `hunt 1,2`
- `gather wood`
- `gather gems`
- `explore`
- `attack` / `guard` / `dodge`
- `cast soultrap`
- `read chronicle_graufurt`
- `book next`
- `lang en` / `lang de`
