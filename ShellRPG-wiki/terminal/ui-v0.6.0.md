# Terminal-UI v0.6.6

Diese Seite dokumentiert die öffentliche Struktur des fünfzeiligen Terminal-Layouts.

## Layout
1. Kontext- / Medienzeile
2. Statuszeile
3. Kampf / Aktion / Route
4. Ereignis- und Systemhinweise
5. Eingabezeile

## Farbidee
- Grau für Stein / Gebirge
- Minzgrün für Untote
- Grün für Orks
- Rot für Blutorks
- Lilarot für Dämonen

## Fallback
Wenn Truecolor nicht verfügbar ist, degradiert das Layout sauber auf einfaches ANSI oder monochrom.
