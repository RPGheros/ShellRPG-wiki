# Client-Shell, Intro und Live-Anzeige v0.7.6

Diese Seite beschreibt die neue Shell-ähnliche Eingabe, den Fake-Boot und die kompakte Live-Anzeige des Terminal-Clients.

## Kernpunkte
- PowerShell-ähnlicher Prompt statt `shellrpg>`
- Shell- und Spielbefehle teilen sich denselben Prompt
- Fake-Boot mit lokalen Geräte-/Protokollhinweisen ohne echte Verbindungen
- Charaktererstellung beim ersten Start
- kompakte Vierzeilen-Live-Anzeige ohne Medienpfad-Spam
- optionale Bildanzeige nur, wenn ein passendes Tool wie `chafa` oder `viu` installiert ist
